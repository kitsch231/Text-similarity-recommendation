# Text-similarity-recommendation
基于文本相似性的一个新闻推荐系统。
采用了tfidf无监督模型和bert模型两种方法。文件限制因此删除了里面的训练数据和bert模型。

tfidf的推荐效果及准确率如下，acc为86。
